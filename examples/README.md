# Examples

Use these prompts to test or demonstrate `super-automation`.

- `Use $super-automation to design an approval workflow between these tools.`
- `Use $super-automation to automate this GitHub issue triage flow.`
- `Use $super-automation to review this workflow for failure modes.`

## Evaluation Checklist

- The skill asks only for missing context.
- The output routes to the correct module.
- The answer is actionable and specific.
- Assumptions are stated clearly.
- Risks, validation steps, or next actions are included where relevant.
